# PRD — 주머니 속 옷장

## 목표

옷 사진을 찍으면 AI가 분석하고, 내 전신 사진 위에 옷을 합성해 피팅하고, 상황별 코디를 추천받는 모바일 앱. YC batch 지원을 위한 MVP — 속도와 안정성의 균형이 핵심.

## 기능 (5개 + 0)

| # | 기능 | 설명 |
|---|------|------|
| 0 | 로그인 | Google OAuth (Supabase Auth 위임). 미로그인 시 sign-in 화면으로 게이팅. 데이터는 Supabase 에 동기화. |
| 1 | 옷 등록 | 촬영/갤러리(최대 5장) → Gemini Flash 분석 → 메타데이터 수정 가능 → 로컬 저장 + Supabase sync |
| 2 | 옷장 | 3열 그리드 + 카테고리 칩 필터 (상의/하의/아우터/원피스/신발/악세서리), 최신순 |
| 3 | 옷 상세 | 메타데이터 편집 (등록 폼 재사용) + 삭제 (soft delete + sync) |
| 4 | 피팅 | 전신 사진(1회 저장, 재사용) + 옷 복수 선택 → Gemini Pro 합성, 결과 저장/바꿔보기 |
| 5 | 스타일 추천 | TPO 6개(출근/데이트/캐주얼/운동/여행/파티) → 메타데이터 기반 Gemini Flash → 조합 3개 + 한줄 코멘트, 입어보기로 피팅 연결 |

> 체형 입력은 삭제 — 전신 사진이 대체. 기능 6개 → 5개로 축소.

## 제약

- ~~DB 없음~~ → Supabase Postgres (Phase 1~5 도입). AsyncStorage 는 로컬 캐시 + push 큐.
- ~~로그인 없음~~ → Google OAuth 로그인 도입. **Apple 로그인 미지원** (별도 마일스톤).
- 타겟: Android + Web + Expo Go (iOS 네이티브 빌드는 별도)
- 부가 스크린 없음 — 화면/UI 요소 최소화
- 포인트 색상 #7A8450 + 무채색만, 이모지 미사용, 단색 아이콘
- service_role key / JWT 검증 secret 은 앱 미포함 — 서버에서만 보유. 앱에는 anon key만.

## 화면 구성

### 내비게이션: 3탭
```
[옷장(홈)]  [피팅]  [추천]
```

### Screen 1: 옷장 (홈)
- 3열 그리드 (사진만, 텍스트 없음)
- 상단: 카테고리 칩 가로 스크롤 (선택 시 #7A8450)
- 우하단: FAB (+) → 바텀시트 (카메라/갤러리)
- 빈 상태: 중앙 (+) + "옷을 등록해보세요"

### Screen 2: 옷 등록
- 카드 리스트 (복수 시 세로 나열)
- 각 카드: 사진 + 카테고리(드롭다운) + 색상(텍스트 칩, 삭제 가능) + 소재(텍스트) + 태그(칩, 삭제/추가)
- 하단 고정: "저장하기 (N벌)"
- 로딩: 전체 화면 스피너 + "분석 중..."

### Screen 3: 옷 상세/수정
- Screen 2와 동일 UI (ClothingForm 컴포넌트 재사용)
- "수정 완료" 버튼 (변경 감지 시 활성)
- "삭제하기" 빨간색 텍스트 → 확인 얼럿

### Screen 4: 피팅
- **미등록 시**: 사람 아이콘 + "전신 사진을 등록해주세요" + 촬영 버튼
- **메인**: 전신 썸네일 + ⚙️(변경) | 최근 피팅 가로 스크롤 | 옷 그리드(체크박스) + 카테고리 필터 | 하단 "피팅해보기 (N벌)"
- **결과**: 합성 이미지 + 사용된 옷 썸네일 + [바꿔보기][저장]
- **촬영**: 사람 실루엣 가이드 오버레이
- **선택 규칙**: 카테고리당 1개, 원피스 선택 시 상의/하의 비활성 (역방향 동일)

### Screen 5: 스타일 추천
- **메인**: "어떤 상황에 입을 옷을 추천해드릴까요?" + TPO 2x3 그리드 (단색 아이콘 + 텍스트)
- **결과**: 조합 카드 3개 (옷 썸네일 나열 + 한줄 코멘트 + "입어보기") + "다시 추천받기"
- "입어보기" → 피팅 메인에 옷 자동 선택 (사용자 확인 후 합성)
- 옷장 부족: 가능한 만큼 + "옷을 더 등록하면 다양한 추천을 받을 수 있어요"

### 공통
- 에러: 토스트 표시
- 로딩: 전체 화면 스피너 + 텍스트
- 빈 옷장 시 피팅/추천: "옷을 먼저 등록해주세요" + 옷장 이동 버튼
