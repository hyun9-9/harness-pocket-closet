# 원본 프롬프트

## 하네스 파이프라인

다음과 같이 작동하는 skill를 만들어라.

1. Clarify
구현가능성 / ux / 데이터베이스 등에 대한 논의점 제안하기.
2. Context Gather
빈 코드베이스 였으므로 필요 x
3. Plan
Task / Phase 파일 만들기
4. Generate
task 실행하기 (run_phases.py)
각 단계별로 별도의 sub agent를 호출하고, 해당 sub agent가 결과를 artifact로 생성하여 다음단계에 전달하도록 구축하자.
5. Evaluate
없음. typecheck,lint,build 등을 추가할수있음

클로드 코드에게 무언가를 시켰을때 잘 못했다면 5개중 무언가를 못했는지 잘 생각 해봐야한다.

## 서비스 요구사항

주머니 속 옷장 — 요구사항
기능 (6개)

옷 등록 — 사진 촬영/선택 -> AI 자동 분석 -> 저장
옷장 — 그리드 목록 + 카테고리 필터
옷 상세 — 정보 확인 + 삭제
체형 입력 — 키/가슴/허리/엉덩이/어깨 (첫 실행 모달) - 필요한 수치 추천해주기
아바타 피팅 — 옷 선택 -> 합성 이미지 + 핏 판정
스타일 추천 — 스타일 선택 -> 조합 3개 + 시각화

구현

앱: React Native (Expo)
서버: FastAPI (Python)
AI: gemini

제약

서버, 데이터베이스를 사용하지 않는다.
로그인을 구현하지 않는다.
부가 스크린을 구현하지 않는다. 화면과 UI요소는 적을 수록 좋다.
포인트 색상 1가지만 제외하고 전부 무채색을 사용한다.

완료 기준
핵심 작동 흐름이 정상 작동한다.
6개 기능 크래시 없이 동작
재실행 후 데이터 유지
API 키 앱에 미포함
